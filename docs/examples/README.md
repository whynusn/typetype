# 代码示例

独立可运行的示例代码应放在此目录，用来展示 TypeType 的最佳实践和常见扩展模式。

---

## 快速导航

| 示例 | 难度 | 说明 |
|:---|:---|:---|
| 暂无独立示例 | - | 当前可参考 `tests/` 中的用例和 `docs/reference/` 的接口速查 |

---

## 示例结构

每个示例应是独立的、可直接运行的代码，包含：

```text
example-name/
├── README.md       # 说明：做什么、怎么运行、学习什么
├── main.py         # 或其他入口文件
├── config.json     # 如需要
└── data/           # 示例数据（如需要）
```

---

## 添加规则

- 每个示例聚焦单一概念，避免复制主项目的大段源码。
- 示例 README 必须包含运行命令、预期输出和相关文档链接。
- API、配置字段、Bridge Slot 的完整规范应指向 `docs/reference/`。
- 架构背景应指向 `docs/ARCHITECTURE.md`。
